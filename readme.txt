IFT3913 - Qualité du logiciel et métriques - Travail pratique 2

Pour ce travail pratique, on a dû étudier un échantillon de données sur les métriques des classes de JFreechart.

À travers ce travail, on a utilisé des fichiers Excel et Java afin de nous aider dans nos calcul

La classe Java principale, EtudeMetrique, nous a servi à prendre les données du fichier jfreechart-stats.csv et à les écrire divisé en rangée dans un fichier appelé etude_metrique.csv, les rendant plus faciles à utiliser pour des calculs.
Le fichier etude_metrique.csv pourrait être converti en fichier de classeur Excel, tel que tp2#1, afin de créer les diagrammes.
De plus, la méthode sondageDCPNOCom dans EtudeMetrique nous a permi de faire le sondage our le numéro 2 en écrivant les données dans un fichier txt appelé tp2#2.

Le fichier tp2#1.csv a été créé afin de pouvoir faire les boîtes à moustache pour le numéro 1.

Les fichiers Excel T3 ont été utilisé pour calculer les rangs et afficher les droites de régression.

Enfin, le fichier Excel T4 a été utilisé pour calculer les coefficients de Spearman.
